COMM 291.1

Numerical Summaries of Center and Spread

Measures of Center

There are three methods to measure the center of a quantitative data set:

  1. Mean — sum of data values divided by number of data values
  2. Median — the middle value after sorting of data
  3. Mode — the highest point in the histogram

The median is a more robust measure than the mean; that is, it is not highly affected by the presence of outliers.

Measures of Spread

The range of a data set is defined as the difference between the maximum value and the minimum value.

A more common measure of spread is variance and standard deviation.

The variance is defined as:

∑(xi + x)2 / n - 1

While the standard deviation is defined as its square root.

In more english terms, the standard deviation is the typical distance of a data value from the mean.

Percentiles

The pth percentile is the value below which p% of the data values fall. There are three important percentiles:

  1. 25th — Q1
  2. 75th — Q3
  3. 50th — median / Q2
