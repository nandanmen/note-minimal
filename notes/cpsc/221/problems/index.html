Prove by induction that a full, complete binary tree of height h has exactly 2h+1-1 nodes.

Trees & Binary Trees

We begin with the base case.

The base case is a tree with a height of 0. We know that this tree has only one node. Inputting 0 to the formula:

20 + 1 - 1 = 1

Thus the formula is true for h = 0.

Inductive Hypothesis

Let it be true for h = k, where 0 < k < h. That is, we assume that a binary tree of height k will have nodes equal to:
